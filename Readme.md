# Gothonweb
A simple text based web game, from Zed Shaw's book "Learn Python the Hard Way".

Screenshot

## Setup
### Install requirements
```bash
pip install -r requirements.txt
```

### Start game
```bash
flask app.py
```

### Run tests
```bash
python -m pytest
```
